# Easy Installer

Testet on:
1. Ubuntu
2. Debian
3. Kali Linux
4. Termux

Available for:
Debian based Linux systems and Termux (Cause use apt)

## Instruction
```bash
apt update && apt upgrade -y
```
By Termux
```bash
pkg install git
```

By Debian based Systems
```bash
apt install git
```

After this
```bash
git clone https://github.com/termuxandlinux/easy-shell
cd easy-shell
chmod +x setup
./setup
```

Then Run
```bash
source $HOME/.bashrc
```

After this Run
```bash
easy-shell
```
